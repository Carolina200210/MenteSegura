# MenteSegura

## Descripción

MenteSegura es un ecosistema digital de salud mental diseñado para facilitar el acceso a recursos de orientación, prevención, atención y acompañamiento psicológico mediante tecnologías accesibles, éticas e inclusivas.

Este repositorio contiene toda la documentación del proyecto, incluyendo investigación, diseño, arquitectura, desarrollo e inteligencia artificial.
